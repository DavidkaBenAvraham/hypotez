Таблица ps_cms_category в PrestaShop обычно используется для хранения информации о категориях CMS (Content Management System). Эти категории используются для организации контента, такого как страницы информации, блоги и другие статические страницы. Вот пример структуры этой таблицы:

| Название поля    | Тип данных    | Описание                              |
|------------------|---------------|---------------------------------------|
| id_cms_category  | INT(10)       | Уникальный идентификатор категории    |
| id_parent        | INT(10)       | Идентификатор родительской категории |
| level_depth      | INT(10)       | Уровень вложенности категории         |
| active           | TINYINT(1)    | Флаг активности категории             |
| date_add         | DATETIME      | Дата и время добавления категории     |
| date_upd         | DATETIME      | Дата и время последнего обновления    |
| position         | INT(10)       | Позиция категории в списке            |
