## ps_product_lang
в PrestaShop 1.7 является частью базы данных и содержит локализованные (языковые) данные для продуктов. Каждая строка в этой таблице соответствует конкретному продукту на определенном языке
### Список полей:
- <b>id_product</b>: Уникальный идентификатор продукта, связанный с основной таблицей ps_product.
- <b>id_shop</b>: Идентификатор магазина, к которому относится запись. <b> Я могу определить, что и в каком магазине продается </b>
- <b>id_lang</b>: Идентификатор языка, для которого предоставлены локализованные данные.
- <b>description</b>: Описание продукта на указанном языке.
- <b>description_short</b>: Краткое описание продукта на указанном языке.
- <b>link_rewrite</b>: Человекочитаемое "чистое" имя URL для продукта.
- <b>meta_description</b>: Мета-описание продукта для поисковых систем.
- <b>meta_keywords</b>: Мета-ключевые слова продукта для поисковых систем.
- <b>meta_title</b>: Мета-заголовок продукта для поисковых систем.
- <b>name</b>: Название продукта на указанном языке.
- <b>available_now</b>: Текст, который отображается, если продукт в наличии.
- <b>available_later</b>: Текст, который отображается, если продукт закончился.
- <b>delivery_in_stock</b>: Текст, который отображается при наличии продукта.
- <b>delivery_out_stock</b>: Текст, который отображается при отсутствии продукта.
- <b>delivery_now</b>: Текст, который отображается, если продукт доступен сразу.
- <b>delivery_later</b>: Текст, который отображается, если продукт будет доступен позже.
- <b>id_product_redirected</b>: Идентификатор продукта, на который происходит перенаправление.
- <b>available_date</b>: Дата начала продажи продукта.
- <b>customizable</b>: Флаг, указывающий, можно ли настраивать продукт (текст, изображение и т. д.).
- <b>uploadable_files</b>: Количество файлов, которые можно загрузить для настройки продукта.
- <b>text_fields</b>: Количество текстовых полей, доступных для настройки продукта.
- <b>active</b>: Флаг активности продукта на указанном языке.
- <b>redirect_type</b>: Тип перенаправления для продукта.
- <b>id_product_redirected</b>: Идентификатор продукта, на который происходит перенаправление.
- <b>available_for_order</b>: Флаг, указывающий, доступен ли продукт для заказа.
- <b>date_add</b>: Дата добавления продукта.
- <b>date_upd</b>: Дата последнего обновления продукта.
