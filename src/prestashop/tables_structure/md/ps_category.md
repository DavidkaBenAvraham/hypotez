Таблица ps_category в PrestaShop используется для хранения информации о категориях товаров. Эти категории используются для организации товаров в магазине. Вот пример структуры этой таблицы:

| Название поля       | Тип данных    | Описание                                        |
|---------------------|---------------|-------------------------------------------------|
| id_category         | INT(10)       | Уникальный идентификатор категории              |
| id_parent           | INT(10)       | Идентификатор родительской категории            |
| level_depth         | INT(10)       | Уровень вложенности категории                   |
| nleft               | INT(10)       | Левая граница дерева категорий                  |
| nright              | INT(10)       | Правая граница дерева категорий                 |
| active              | TINYINT(1)    | Флаг активности категории                       |
| date_add            | DATETIME      | Дата и время добавления категории               |
| date_upd            | DATETIME      | Дата и время последнего обновления категории    |
| position            | INT(10)       | Позиция категории в списке                      |
