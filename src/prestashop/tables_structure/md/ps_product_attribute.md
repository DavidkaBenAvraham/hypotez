## ps_product_attribute

В PrestaShop 1.7, таблица ps_product_attribute содержит информацию о комбинациях атрибутов для продуктов. Эти комбинации могут представлять собой различные варианты продукта, такие как разные размеры, цвета и т. д. Вот основные поля, которые содержатся в этой таблице:
### Поля:
- <b>id_product_attribute</b>: Уникальный идентификатор комбинации атрибутов продукта.
- <b>id_product</b>: Идентификатор основного продукта.
- <b>reference</b>: Артикул (код) для комбинации атрибутов.
- <b>supplier_reference</b>: Артикул поставщика для комбинации атрибутов.
- <b>location</b>: Местоположение комбинации атрибутов на складе.
- <b>ean13</b>: Штрих-код для комбинации атрибутов.
- <b>isbn</b>: Международный стандартный номер книги (ISBN) для комбинации атрибутов.
- <b>upc</b>: Универсальный код продукта для комбинации атрибутов.
- <b>mpn</b>: Номер производителя для комбинации атрибутов.
- <b>wholesale_price</b>: Оптовая цена для комбинации атрибутов.
- <b>price</b>: Цена для комбинации атрибутов.
- <b>ecotax</b>: Экологический налог для комбинации атрибутов.
- <b>quantity</b>: Количество доступных продуктов для комбинации атрибутов.
- <b>weight</b>: Вес для комбинации атрибутов.
- <b>unit_price_impact</b>: Влияние на цену единицы товара.
- <b>default_on</b>: Флаг, указывающий, является ли комбинация атрибутов атрибутом по умолчанию.
- <b>minimal_quantity</b>: Минимальное количество для заказа комбинации атрибутов.
- <b>low_stock_threshold</b>: Порог низкого уровня запасов для комбинации атрибутов.
- <b>low_stock_alert</b>: Флаг предупреждения о низком уровне запасов для комбинации атрибутов.
- <b>available_date</b>: Дата начала продажи для комбинации атрибутов.
- <b>date_upd</b>: Дата последнего обновления комбинации атрибутов.
Этот список полей может быть расширен или изменен в зависимости от настроек и установленных модулей в вашей установке PrestaShop.
