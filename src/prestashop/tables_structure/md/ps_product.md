### ps_product
одна из основных таблиц базы данных, которая хранит основную информацию о товарах в вашем магазине
### Список полей:
- <b>id_product</b>: Уникальный идентификатор товара.
- <b>id_supplier</b>: Идентификатор поставщика товара.
- <b>id_manufacturer</b>: Идентификатор производителя товара.
- <b>id_category_default</b>: Идентификатор основной категории, к которой принадлежит продукт.
- <b>id_shop_default</b>: Идентификатор основного магазина, связанного с продуктом.
- <b>id_tax_rules_group</b>: Идентификатор группы налоговых правил, применяемых к продукту.
- <b>on_sale</b>: Флаг, указывающий, находится ли продукт на распродаже.
- <b>online_only</b>: Флаг, указывающий, доступен ли продукт только онлайн.
- <b>ean13</b>: Штрих-код товара (если установлен).
- <b>upc</b>: Универсальный код товара (если установлен).
- <b>ecotax</b>: Экологический налог на продукт.
- <b>quantity</b>: Доступное количество товара на складе.
- <b>minimal_quantity</b>: Минимальное количество, которое можно заказать.
- <b>price</b>: Цена товара.
- <b>wholesale_price</b>: Оптовая цена товара.
- <b>unity</b>: Единицы измерения товара.
- <b>unit_price_ratio</b>: Коэффициент преобразования для цены за единицу.
- <b>additional_shipping_cost</b>: Дополнительная стоимость доставки.
- <b>reference</b>: Артикул или код товара.
- <b>supplier_reference</b>: Артикул или код товара поставщика.
- <b>location</b>: Местоположение товара на складе.
- <b>width, height, depth, weight, volume</b>: Габариты объем и вес товара.
- <b>out_of_stock</b>: Поведение, когда продукт заканчивается (например, "Отображать когда нет в наличии" или "Скрыть, когда нет в наличии (1 - нет в наличии, 0 - есть) ").
- <b>quantity_discount</b>: Флаг, указывающий, действуют ли скидки от объема.
- <b>customizable</b>: Флаг, указывающий, можно ли настраивать продукт.
- <b>uploadable_files</b>: Количество файлов, которые можно загрузить для настройки товара.
- <b>text_fields</b>: Количество текстовых полей, доступных для настройки товара.
- <b>active</b>: Флаг активности товара.
- <b>redirect_type</b>: Тип перенаправления для товара.
- <b>id_product_redirected</b>: Идентификатор товара, на который происходит перенаправление.
- <b>available_for_order</b>: Флаг, указывающий, доступен ли продукт для заказа.
- <b>available_date</b>: Дата начала продажи товара.
- <b>show_condition</b>: Флаг, указывающий, отображать ли условие товара.
- <b>condition</b>: Условие товара (новый, б/у и т.д.).
- <b>sho
- <b>indexed</b>: Флаг, указывающий, проиндексирован ли продукт для поисковых систем.
- <b>visibility</b>: Уровень видимости товара.
- <b>cache_default_attribute</b>: Кэш-значение для атрибутов товара.
- <b>advanced_stock_management</b>: Флаг, указывающий, используется ли продвинутое управление запасами.
- <b>date_add</b>: Дата добавления товара.
- <b>date_upd</b>: Дата последнего обновления товара.
